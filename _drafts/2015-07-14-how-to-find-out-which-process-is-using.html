On Linux (and probably other UNIX-like operating systems you can do the following:

lsof -i :8080


On Windows you can do the following (well, assuming that you have the grep available, for instance installing Git Bash and adding it's cmd directory to your PATH environment variable):

netstat -nabo | grep :8080
