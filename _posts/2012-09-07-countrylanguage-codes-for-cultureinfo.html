As far as I know, there is no such page in MSDN. However, for the record, the code is composed by a two letter lower case ISO code for language (e.g. "pt", "en") and two letter upper case ISO code for country (e.g. "PT", "US"), separated by the hyphen character ("-"). Both of these codes are documented in Wikipedia as ISO 3166-1 alpha 2 and ISO 639-1, respectively.
