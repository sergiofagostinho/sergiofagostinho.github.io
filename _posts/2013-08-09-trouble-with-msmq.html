The issue was that I was sending messages using the MessageQueue class, but the messages didn't get there. No exception, no error code, nothing in the Event Viewer. Then I started "mmc", and added the "Computer Management" snap-in, and navigated to Services and Applications > Message Queuing. There I found that all my queues were OK (Private Queues), but no message was getting there. Then I found some errors in the "Outgoing Queues" folder, telling me that the queue did not exist.
So, what was the issue? Well, it happens that the NetBIOS name on my machine did not match my computer name. Our machine names are based on a convention which includes the  user surname, and mine is a little bit longer that most my colleague developers, which goes over the 15 character limit for NetBIOS naming conventions. Probably the administrator that configured my machine just cut the last character from the NetBIOS name, and never looked back again...
