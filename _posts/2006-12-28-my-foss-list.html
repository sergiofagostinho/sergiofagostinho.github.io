FOSS who?

Free or Open-Source Software (FOSS). That means software that you have at least access to its source code, and depending on its license more or less freedom to use, modify and redistribute. Not to be mistaken with freeware, software at no cost. Although usually free software is also free in terms of price, it is above all free in terms of freedom of use. The classical example, is free as in free beer versus free as in freedom of speech.

Licenses and platforms

The platform is important to know if you can use the software:
  • Win32: any 32 bit MS Windows operating system (Windows family 9x: 95, 98, ME; Windows family NT: NT 4, 2000, XP, 2003) and probably the most recent operating systems (Windows XP and 2000 64 bit; Windows Vista).
  • Linux: any GNU/Linux distribution, with the usual library dependencies.
  • OS X: any Apple MacOS X system.
  • *BSD: any BSD based operating system (FreeBSD, NetBSD and OpenBSD).
  • Solaris: any Sun Solaris operating system (further restrictions apply).
  • Java: any operating system with a Java Virtual Machine installed (Sun supports Linux , Windows and Solaris, but third-party vendors offer it to other operating systems).
  • other: I would not believe that there are still people coding software for OS/2 and BeOS, but it is true.
If you are not a computer expert, do not mind the license. If you are interested though, most software is in BSD (no limitations at all), GPL (can not change the license or compile with other license software, among other restrictions), or somewhere in between (LGPL, EPL, MPL, etc).
